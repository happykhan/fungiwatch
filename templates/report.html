Methods

Species selection

The 19 fungal entities from the WHO Fungal Priority Pathogens List (2022) are queried, grouped into Critical (4), High (7), and Medium (8) priority. For genus-level entries (e.g. Histoplasma spp.), the genus name is queried. For Mucorales, the order name is used. For eumycetoma agents, multiple causative genera are queried (Madurella, Medicopsis, Falciformispora, Trematosphaeria). Where NCBI taxonomy has been updated (e.g. Candida glabrata → Nakaseomyces glabrata), both old and new names are queried and results deduplicated by accession.

Assembled genomes

Genome assembly metadata is retrieved using the NCBI Datasets CLI (datasets summary genome taxon). Fields extracted per assembly: accession, organism name, taxonomy ID, release date, assembly level, genome size (bp), GC content (%), and BioSample attributes (collection date, geographic location). Both GenBank and RefSeq assemblies are included.

SRA WGS runs

Sequencing read metadata is retrieved from the NCBI Sequence Read Archive (SRA) via E-utilities (esearch + efetch). Runs are filtered to library_source=GENOMIC and library_strategy=WGS to include only whole-genome sequencing data. BioSample attributes (collection date, geo_loc_name) are extracted from the inline sample metadata in the SRA XML. Placeholder values ("missing", "not collected", "not applicable", "unknown") are treated as absent.

Metadata completeness

For each record (genome or SRA), "With Location" indicates a non-empty geo_loc_name BioSample attribute; "With Date" indicates a non-empty collection_date; "With Both" requires both fields present. The data availability table can be filtered by source (Genomes only, SRA only, or combined).

Visualisations

Geographic distribution: The geo_loc_name field is parsed to extract the country (text before the first colon), mapped to ISO 3166-1 alpha-3 codes, and displayed on a Natural Earth 110m equirectangular projection with a log-scale colour gradient. Combined genome + SRA data.

Published over time: Uses genome assembly release dates and SRA run publication dates, stacked by priority group or species.

Genome size vs GC content: Assembled genomes only — SRA runs lack assembly-level statistics.

Sample collection timeline: Uses BioSample collection_date from both genomes and SRA runs, showing when isolates were originally sampled.

Limitations

  • SRA counts for genus-level queries may include non-pathogenic species within the genus.
  • Geographic parsing is approximate — entries with only sub-national locations (e.g. "California" without country prefix) may be missed.
  • Collection dates vary in precision (year only, year-month, or full date); only the year is used for temporal charts.
  • Some assemblies in NCBI are suppressed or superseded but still returned by the Datasets CLI.
